# [Print Elements of Array](https://www.geeksforgeeks.org/problems/print-elements-of-array4910/1?page=1&category=Arrays,Java&difficulty=Basic&sortBy=submissions)
## Easy
Given an array arr[], print all its elements space-separated.
Note: You don't need to move to the next line after printing all elements of the array (space-separated)
Examples:
Input: arr[] = [1, 2, 3, 4, 5]
Output: 1 2 3 4&nbsp;5

Input: arr[] = [2, 3, 5, 5]
Output: 2 3 5 5

Constraints:1 &lt;= arr.size &lt;= 1061 &lt;= arr[i] &lt;= 108

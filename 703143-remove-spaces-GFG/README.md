# [Remove Spaces](https://www.geeksforgeeks.org/problems/remove-spaces0128/1?page=1&category=Strings,Java&difficulty=Basic&sortBy=submissions)
## Easy
Given a string s. Your task is to remove spaces from it.&nbsp;
Examples:
Input: s = "geeks&nbsp; for geeks"
Output: "geeksforgeeks"
Explanation: All the spaces have been removed.

Input: s = "&nbsp;&nbsp;&nbsp; g f g"
Output: "gfg"
Explanation: All the spaces including the leading ones have been removed.

Constraints:1&lt;=|s|&lt;=105

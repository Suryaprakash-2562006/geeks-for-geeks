# [Greatest of three numbers](https://www.geeksforgeeks.org/problems/greatest-of-three-numbers2520/1?page=1&category=Java&status=unsolved&sortBy=submissions)
## Easy
Given 3 numbers a, b and c. Find the greatest number among them.
Examples:
Input: a = 10, b = 3, c = 2
Output: 10
Explanation: 10 is greatest among the three 
Input: a = -4, b = -3, c = -2
Output: -2
Explanation: -2 is greatest among the three
Constraints:-109≤ a, b, c ≤109

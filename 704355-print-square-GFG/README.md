# [Print Square](https://www.geeksforgeeks.org/problems/print-square/1?page=3&category=Java&sortBy=submissions)
## Easy
Given an integer s, write a program to print the square of size s&nbsp;using "*"&nbsp;character.&nbsp;Note: Make sure to add a " " between two "*". Add a new line after printing the square
Examples : 
Input: s = 4
Output:
* * * **     **     ** * * *
Explanation: It's a square! Each side contains s = 4 *.
Input: s = 3
Output:
* * * *   ** * *
Explanation: It's a square! Each side contains s = 3 *.
Constraints:1 ≤ s ≤ 10

# [Greatest of Three](https://www.geeksforgeeks.org/problems/greatest-of-three--154612/1?page=3&category=Java&difficulty=Basic,Easy&sortBy=submissions)
## Easy
Given three numbers a, b, and c. You need to find which is the greatest of them all.
Examples:
Input: a = 1, b = 2, c = 3
Output: 3
Explanation: Clearly, c = 3 is the greatest of (1, 2, 3)
Input: a = 2, b = 2, c = 5
Output: 5
Explanation: Out of (2, 2, 5) 5 is the greatest.
Constraints:1 &lt;= a, b, c &lt;= 103

# [For Loop - 2](https://www.geeksforgeeks.org/problems/for-loop-2--182531/1?page=3&category=Java&difficulty=Basic&sortBy=submissions)
## Easy
You are given a string s, you need to print its characters at even indices (index starts at 0). 
Examples: 
Input: s = "Geeks"
Output: Ges
Explanation: The even indices characters are printed.
Input: s = "DoctorPhenomenal"
Output: DcoPeoea
Explanation: The even indices characters are printed.

# [Count Digits](https://www.geeksforgeeks.org/problems/count-digits-1606889545--123832/1?page=7&category=Java&sortBy=submissions)
## Easy
Given a natural number n. You have to find the number of digits in it and return it.
Examples:
Input: n = 5534
Output: 4 
Explanation: 5534 has 4 digits
Input: n = 100273
Output: 6
Explanation: 100273 has 6 digits
